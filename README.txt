Étapes pour ouvrir un projet avec XAMPP et htdocs:

1. Télécharger XAMPP
Ouvrir le site officiel de XAMPP :Apache Friends - XAMPP


Cliquer sur la version adaptée à votre système :Windows/Linux/macOS


Télécharger l’installateur.



2. Installer XAMPP
Ouvrir le fichier téléchargé.


Cliquer sur : Next


sélectionner les composants nécessaires :Apache/MySQL/PHP


puis continuer avec Next


Choisir le dossier d’installation : C:\xampp


Cliquer sur Install.


Attendre la fin de l’installation.


Cliquer sur Finish.



3. Ouvrir XAMPP Control Panel
Après l’installation : Rechercher : XAMPP Control Panel


L’ouvrir.



4. Démarrer Apache 
Dans le panneau XAMPP :Cliquer sur Start devant :Apache et SQL


La ligne devient verte : Running


⚠️ Si Apache ne démarre pas :fermer Skype/IIS       changer les ports 80 ou 443



5. Aller dans le dossier htdocs
Le dossier principal des projets se trouve ici :
C:\xampp\htdocs

6. Copier le projet dans htdocs
Exemple :C:\xampp\htdocs\StayEase


7. Vérifier le fichier principal
Le fichier principal doit s’appeler :index.html


8. Ouvrir le navigateur
Accéder au projet: Dans la barre d’adresse : http://localhost/StayEase/index.html
